# TimeNugget Docs

> 大学生高效利用时间的经验分享文档

## 简介

TimeNugget 是一个专为大学生打造的时间管理和学习经验分享平台，帮助大学生避坑、快速上手大学生活。

## 快速开始

### 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建文档
npm run docs:build

# 预览构建结果
npm run docs:preview
```

## 技术栈

- [VitePress](https://vitepress.dev/) - 静态文档生成器
- Markdown - 文档编写
